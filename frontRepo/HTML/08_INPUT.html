라디오 버튼

        라디오 버튼은 같은 name을 줘야 하나의 그룹으로 묶인다.
        라디어 버튼은 하나의 값만 선택하게 할 때 사용한다. (다중선택 X)
        기본값으로 체크가 되어있게 하려면, checked 속성 부여
        submit시, check된 input태그의 value가 전달된다.
        value가 없으면 on이라는 값으로 전달된다.


체크박스

          체크박스는, 다중선택을 할 때 사용한다.
          체크박스 선택 시 해당 태그의 value가 전달된다.
          name 값은 똑같이 주면 된다. (어차피 전부 넘어간다.)
          기본적으로 체크될 요소들을 지정할 수 있다. (checked)

버튼 태그


      
        일반버튼 / 서브밋버튼 / 리셋버튼
        폼태그 밖에서는 일반 버튼,
        폼태그 내의 버튼은 제출 역할을 한다.
        form태그 내에서 버튼을 그냥 만들면, type이 submit으로 지정된다.
        다르게 사용하고 싶으면 type 따로 지정해줘야 한다.

        리셋은, form태그 내의 input태그들의 value를 초기화한다.
      


      

        
        

        
        

        
        

        
        

        
        

        
      


      

      


      


      
셀렉트 박스


      
        셀렉트 박스는,
        목록에서 값을 선택해서 서버로 전달한다.
        select - option은
        option태그의 content영역에
        서버로 넘길 데이터를 넣어줄 수 있다.
        (value속성을 이용해 지정도 가능)

        option태그에 selected 속성을 추가하면 기본적으로 선택이 된다.
