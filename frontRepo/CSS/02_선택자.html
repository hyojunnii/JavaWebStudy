선택자

요소를 선택하는 방법

        * : 모든 요소 선택
        # : id로 요소 선택
        . : class로 요소 선택
        태그명을 이용해서 선택
        ","를 이용해서 여러개 동시 선택 가능

        우선순위
        id > class > tagname

        클래스는 띄어쓰기를 이용해서
        여러개를 부여할 수 있다.

        클래스와 아이디를 동시에 부여할 수 있다.

        -클래스 선택자 : 
        다른 태그와 중복가능

        -아이디 선택자 : 
        고유한 값 작성할 것
