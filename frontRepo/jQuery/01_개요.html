jQuery 개요

jQuery 란?

JS만 이용해서 구현하기 복잡하고 어려운걸 처리하기 위해 등장
존 레식이 JS를 편하게 사용하라고 만들어놓은 라이브러리
JS를 이용해서 만든 라이브러리 (라이브러리 까보면, 결국 JS)
복잡한 JS 구문을 짧고 쉽게 개발 가능

jQuery 장점

1) DOM 조작 쉬움
2) AJAX 지원
3) 플러그인, 오픈api 지원 (차트, 슬라이드 등..)

jQuery 연결 방법

1) 파일
2) 링크 (CDN 방식)
